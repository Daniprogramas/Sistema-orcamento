# Sistema-orcamento
 Sistema Web de Orçamentos

Sistema Web para criação, gerenciamento e visualização de orçamentos, desenvolvido como projeto acadêmico com foco em boas práticas de desenvolvimento frontend, integração com API REST e organização profissional de código, alinhado aos requisitos do Processo Seletivo NUPLAM+DIGITAL (IMD/UFRN).

 Objetivo do Projeto

O objetivo deste projeto é demonstrar competências técnicas e comportamentais exigidas para a vaga de Desenvolvedor Web Júnior, por meio do desenvolvimento de um sistema realista, inspirado em demandas do contexto industrial, como controle de orçamentos e valores.

O projeto enfatiza:

Desenvolvimento frontend moderno
Consumo de APIs REST
Organização, clareza e documentação
Uso adequado de controle de versões (Git/GitHub)
